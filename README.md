# UK Nighttime Economy

## Brief

This is a Python dashboard for visualising UK nighttime economy. You can view this web-app at <fill this>.

Please follow the instruction for installation and running it locally. 

## Environment

- Python 3.12 with appropriate `pip`.
- Python Libraries / Packages:
  - NumPy 2.4.2
  - Pandas 2.3.3
  - Plotly 6.5.0
  - Streamlit 1.54.0

This project was tested in the macOS Monterey 12.6, Windows 10, and Ubuntu 20.04.

## Running locally

1. Locate to the root of this project `cd uk-nighttime-economy`

2. Install the Python libraries and packages and the associated dependencies `pip install -r requirements.txt`

3. Start the streamlit web server locally `streamlit run app.py`

4. Navigate to http://localhost:8501/ in your browser.
